void CommandExecutor::tryExecute ( const size_t &  commandId,

Running on it's own thread. Checks if the handles the command requires are occupied. If so, call interrupt() on all command's currently using the handles the new command requires. After that waits for the condition variable to be triggered to then check again. If all handles are free, execute the command in the current thread. Sets status to started.
