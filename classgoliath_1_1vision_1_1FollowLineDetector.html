Detailed Description

Takes in an input image, splits it up into "boxes". Within those boxes it checks if there is a single contour, if so take the center of that contour and compare it to the center of the box. This will return an offset. Take the average of all offsets to determine if the robot should go straight, left or right. The second value will be the distance from the edge of the box (percentage) the robot will have to turn.
